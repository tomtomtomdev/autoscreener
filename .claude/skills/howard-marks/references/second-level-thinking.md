# Second-Level Thinking

> "First-level thinking says, 'It's a good company; let's buy the stock.' Second-level thinking says, 'It's a good company, but everyone thinks it's a great company, and it's not. So the stock's overrated and overpriced; let's sell.'" — Howard Marks, *The Most Important Thing*

## The core idea

Markets aggregate the views of millions of participants into the price. So to beat the market, being *right* isn't enough — you have to be right in a way that **differs from the consensus**, because the consensus is already in the price. Marks frames this as the only path to superior returns: a non-consensus view that turns out to be correct.

This gives you a 2×2:

|                          | You're right        | You're wrong          |
|--------------------------|---------------------|-----------------------|
| **Consensus agrees**     | average return (priced in) | average loss |
| **You differ from consensus** | **superior return** | painful underperformance |

Superior performance only lives in the bottom-left box. And to get there you have to be willing to look wrong/foolish for a while — to "be different and lonely," as Marks puts it. The asymmetry is brutal: being non-consensus is necessary but not sufficient; you also have to be correct, and most attempts to be different are just wrong.

## How to apply it

When the user pitches a thesis (or you're evaluating one), don't just ask "is this true?" Ask the chain:

1. **What is the consensus view here?** What does the price imply the market already believes about this asset's future?
2. **How does this thesis differ from that consensus?** If it doesn't — if the thesis is "it's a great company with growing earnings" and everyone agrees — then even if it's *true*, there's likely no edge. It's priced in.
3. **If it does differ, why might the crowd be wrong?** Marks's legitimate sources of edge: superior information (rare/fleeting), superior analysis of the same information, or superior behavior/temperament (most durable). What's the source here?
4. **What would have to be true for the variant view to pay off, and how would you know if you're just wrong?**

## The trap to flag

The most common failure: mistaking a *correct first-level observation* for an *edge*. "NVIDIA makes the best AI chips" can be completely true and completely useless as an investment thesis, because it's universally known and embedded in a high price. Push the user from "is this a good company?" to "is the gap between this company's quality and its price mispriced relative to what everyone already believes?"

Conversely, watch for the inverse: a contrarian view that's contrarian *because it's wrong*. Differing from the crowd is not itself virtuous (see `psychology-and-pitfalls.md` on "contrarianism isn't a slogan"). The bar is non-consensus **and** demonstrably correct.

## Output pattern

State the consensus, state how the thesis diverges (or admit it doesn't), name the proposed source of edge, and give an honest read on whether there's genuine second-level insight or just a well-articulated first-level take. "This is a sound observation but it's the consensus — I don't see the edge" is a valuable and frequently correct verdict.
